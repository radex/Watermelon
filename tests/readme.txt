Contents of this folder are PHPUnit test cases. Hiearchy is meant to be very similar of that in wmelon/.

To run tests cd to this folder and run "phpunit ."

Some tests uses special database watermelon_tests on localhost using user watermeloner with password wtrmln123. To install necessary tables, import wmelon/bundles/installer/sql/structure.sql and tests/sql.sql into watermelon_tests database.
